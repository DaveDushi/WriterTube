## Introduction

This web application is designed to take a YouTube link as input and generate an article based on the content of the video. Leveraging the power of OpenAI and AssemblyAI, it provides a convenient way to convert video content into readable articles.

## Setup

### Prerequisites

Before setting up the application, ensure you have the following installed:
- Supabase account (for hosting the PostgreSQL database)
- OpenAI API key
- AssemblyAI API key
